# jianzhunenghao
建筑节能监测系统+公共建筑能耗监测系统+建筑能耗监测系统+节能监测系统+能耗监测+建筑能耗监测+能耗分析+能耗管理+能耗预测+能耗监控+能耗监测平台+建筑能耗
# 介绍
 能耗监控系统是为耗电量、耗水量、耗气量（天然气量或者煤气量）、集中供热耗热量、集中供冷耗冷量与其他能源应用量的控制与测量提供解决方案的能耗监控系统。
